# Rayyan Sarwar Syed — Portfolio Website

Live site: https://rayyansarwarsyed.github.io/Portfolio/

## About This Project

This is my personal portfolio website built to showcase my work in **Technology Operations, Cybersecurity, Network Support, SOC Readiness, Secure Workflows, and Technical Documentation**.

The goal of this portfolio is to go beyond a traditional resume and create a living proof system where my experience, projects, skills, competitions, certifications, and technical growth are organized in one place.

This site highlights my ability to support users, troubleshoot systems, understand security workflows, analyze technical problems, and document work clearly for teams.

## What This Portfolio Includes

- Professional summary and career focus
- Technology operations and cybersecurity value areas
- Interactive story section connecting support, security, networking, and documentation
- Experience timeline
- Technical workflow breakdown
- Project roadmap and case study cards
- Skills explorer
- Competitions, certifications, and education section
- Resume, LinkedIn, GitHub, and email links
- Responsive design for desktop and mobile
- Dark/light theme toggle
- Interactive UI elements using vanilla JavaScript

## Main Focus Areas

### Technology Operations

This portfolio reflects my experience supporting users, troubleshooting access issues, working with ticketing systems, documenting technical problems, and helping teams stay productive.

### Cybersecurity

The site highlights my interest and training in SOC readiness, alert triage, phishing analysis, vulnerability review, suspicious login behavior, SIEM/log review, and incident documentation.

### Networking

Networking is a key part of my troubleshooting mindset. The portfolio includes work connected to DNS, DHCP, TCP/IP, LAN/WLAN, packet analysis, Wireshark, Nmap, and exposed services.

### Documentation

A major theme of this portfolio is turning technical work into reusable documentation, including ticket notes, escalation summaries, README files, investigation notes, and remediation plans.
